--- layout: post title: C# on the journey to functional programming date: '2008-01-03T20:15:00.001+01:00' author: Jonas Bandi tags: modified_time: '2008-01-03T22:10:41.794+01:00' blogger_id: tag:blogger.com,1999:blog-5763764290649132593.post-1989244330065103122 blogger_orig_url: http://blog.jonasbandi.net/2008/01/c-on-journey-to-functional-programming.html --- From old-school imperative... 
var list = new List<int>(){1, 2, 3};
int sum = 0;
foreach (int n in list)
{
   sum += n;
}
Assert.AreEqual(6, sum);
...over clumsy delegate... 
var list = new List<int>(){1, 2, 3};
int sum = 0;
list.ForEach(delegate(int number) { sum += number; });
Assert.AreEqual(6, sum);
...to sexy lambda. 
var list = new List<int>(){1, 2, 3};
int sum = 0;
list.ForEach(number => sum += number);
Assert.AreEqual(6, sum);
