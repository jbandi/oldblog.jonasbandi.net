--- layout: post title: First steps in creating an internal DSL with Groovy date: '2009-08-05T20:32:00.001+02:00' author: Jonas Bandi tags: - groovy - programming - javablog - DSL modified_time: '2009-08-05T20:33:55.967+02:00' blogger_id: tag:blogger.com,1999:blog-5763764290649132593.post-5992117434122733280 blogger_orig_url: http://blog.jonasbandi.net/2009/08/first-steps-in-creating-internal-dsl.html --- Sten Anderson has a great post about creating an internal DSL with Groovy: A Text Adventure DSL in Groovy.

The "domain" of this example DSL is old-school adventure games like King's Quest.

The post shows the different steps how to transform the following Java-like code... 
game.look();
game.go("north");
game.take("dagger");
game.take("sword");
game.take("key");
game.take("axe");

... into the following statements: 
look
go north
take dagger
grab sword
hold key
yoink axe

Both of the above snippets are valid Groovy code that can be executed on the JVM!

To understand and learn about the implementation of internal DSLs I implemented this DSL example.

The project can be browsed and downloaded on my github account. The project is fully mavenized, and should easily be buildable and runnable with Maven.
