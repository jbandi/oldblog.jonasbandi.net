--- layout: post title: Useful maven commands date: '2008-05-28T16:42:00.001+02:00' author: Jonas Bandi tags: - programming modified_time: '2008-05-30T12:22:01.053+02:00' blogger_id: tag:blogger.com,1999:blog-5763764290649132593.post-647652330488400152 blogger_orig_url: http://blog.jonasbandi.net/2008/05/useful-maven-commands.html --- To skip tests, you can set the property maven.test.skip=true, i.e:
mvn -Dmaven.test.skip=true package


To list all the dependencies of your project, you can use the dependency plugin, i.e:
mvn dependency:tree


To build a web-site with a lot of information about your project, you can use the site plugin, i.e:
mvn site

This generates a directory 'site' inside your 'target'.
