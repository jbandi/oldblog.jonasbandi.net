Currently there seems to be a second wave of cloud platforms.

The first wave were:

Google App Engine (GEA)

Amazon EC2

Windows Azure

VMWare / SpringSource

Rackspace http://www.rackspace.com/cloud/

 

These are the big players in the industry.

 

The second wave now sits on top of these platforms. Interestingy there seem to be small companies:

- Heroku (for Ruby)

- CloudBees (for Java)

- AppHarbor (for .NET)

- Moncai

- http://www.springsource.com/code2cloud

 

This second wave does not only focus on scalability but it focuses on the whole deployment and application lifecycle management topic.

They try to promote a workflow that takes place fully in the cloud. Not only the hosting of the productive app is happening in the cloud but also the other phases of the application lifecycle:

Source control in the cloud

Build services / continuous integration in the cloud

Deployment happening in the cloud

Seamless integration of deployment, development and source control
